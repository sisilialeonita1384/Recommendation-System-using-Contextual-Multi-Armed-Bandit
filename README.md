# Recommendation-System-using-Contextual-Multi-Armed-Bandit

This project is a DASH diet-based menu recommendation system designed to help manage hypertension through proper nutrition. It uses a Contextual Multi-Armed Bandit approach that learns from user ratings to continuously improve and personalize meal recommendations based on nutritional needs and preferences.

How to run:
1. Run scraping_resep.ipynb
2. Run scraping_buah.ipynb
3. Run mapping_sheets.ipynb
4. Run preprocessing_data.ipynb
5. Run pembentukan_arm.py
6. Run evaluasi_cmab_realuser.py
